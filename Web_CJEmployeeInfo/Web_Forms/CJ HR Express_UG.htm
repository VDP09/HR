Overview:                                                                                                   á

 

This new system will replace all employee status change forms, which are filled out at the store and sent to the Main Office. This system will work as a single point of entry for all HR and Payroll related employee changes. These changes will flow through several approval points including Ceridian (Payroll system), the Aloha system at the store, a new employee database at the Main Office, and an Email system. The following employee changes will be entered through this system:

 

  • New Hire / Rehire
  • Employee Name /Address / Contact Changes
  • Department / Pay Rate Changes
  • Permanent Employee Transfers
  • Employee Separation

 

 

·         There is no need to send any Status Change forms to the Main Office once this system is in place.

 

·         All requests made through this system (except Name/Add Change) will first go in the queue for GM’s approval and then in the queue for main office. Any user (OA or Managers) error can be corrected by GM/AGM and if missed, by main office user.

 

·         The GM must make all status change requests for salaried employees.

 

·         This system uses Windows Security System to identify the user. The user must login to Windows by his/her own Windows login. General Windows account CJxx will not work.

 

·         The link for “HR Express” can be found on the MineShaft main page under the section labeled Web Forms.

 

 

 

Main Window:                                                                                           á

 

 

 

·         This is what the main screen looks like. The OA, Managers & AGM will be able to see all hourly active employees of her/his store. The GM will be able see all active employees of his/her store.

 

·         You can search for an employee by selecting the Department or by entering specific Search Criteria (First Name, Last Name, Clock Id, SSN). Click Get Employees and a list of employees will appear in the grid. Select the employee you want to enter a status change request for by selecting the row with mouse.

 

·         After the appropriate employee is selected, click on any of the options below and a new window will pop up where you can enter information and submit your requests.

 

·         To create a request for a new hire or rehire, click the button New Hire / ReHire. A New Hire / ReHire window will pop up where you can submit a new hire or rehire request. Upon successfully submitting a new hire request, the new hire Clock Id will appear on the same window.

 

·         Click on Quick Hire if you don’t have all new hire / rehire information. A new window will pop up where you can input limited information and submit. Upon successfully submitting a request, the new hire Clock Id will appear on the same window. All new hire requests will go to Update Requests queue, from where OA will have to update the request with other necessary data.

 

 

New Hire / ReHire window:                                                                      á

 

 

 

·         Use this window to request a new hire or rehire.

 

·         All fields with pink color are mandatory.

 

·         Send new hire paperwork like application, I-9, W-4, SSN, Driver Lic. Etc. to Main Office through weekly mail.

 

·         Aloha system of your store will be updated automatically with the new hire / rehire information the next day early morning.  Employee will be able to clock in the next day.

 

·         If you want the employee to start work the same day, get employee’s Clock Id (which will appear on the same window, once you successfully submit the request), and their name and update Aloha system manually so employee can clock in. Employee’s other information like address, wage, etc. will be updated in your Aloha system the next day automatically.

 

·         It’s recommended you make new hire / rehire requests one day prior to the start of first working day.

 

 

 

Quick Hire :                                                                                               á

 

 

·         Use this window to request a quick new hire or rehire with limited information.

 

·         All fields in pink color are mandatory.

 

·         All quick hire requests will go to Update Requests queue, where OA will have to update with other necessary data.

 

·         Aloha system of your store will be updated automatically with the Quick Hire information the next day early morning.  Employee will be able to clock in the next day.

 

·         If you want the employee to start work the same day, get employee’s Clock Id (which will appear on the same window, once you successfully submit the request), and their name and update Aloha system manually so employee can clock in. Employee’s other information like address, wage, etc. will be updated in your Aloha system the next day automatically.

 

·         It’s recommended you make new hire / rehire requests one day prior to the start of first working day.

 

 

 

 

Department / Pay Rate Change window:                                                  á

 

 

  • Use this window to do the following…

 

·         Add new Department

·         Remove existing Department

·         Change Pay Rate for any existing Department

·         Change Primary Department

 

  • Only one department can be primary.

 

  • To add new Department, select department from the list, enter pay rate and click << Add.

 

  • To remove department, select department from the Employee’s new department / Pay Rate and click Remove Department.

 

  • To change pay rate for any existing department, remove department first and then add the same department with a new pay rate.

 

  • To change primary department, remove primary department from the list and add new department with Primary box checked.

 

  • Effective Date must be either the 1st or 16th of the month.

 

  • If employee is not available to sign, write “n/a” on signature pad.

 

  • Once this request is approved by main office, all department/pay rate changes will be updated to your Aloha system automatically.

 

 

 

Name / Address Change window:                                                             á

 

 

 

·         Use this window to change employee’s name, address or phone contact information.

 

·         Fields in yellow color are mandatory.

 

·         Enter employee’s new information under Employee’s New Information section and click Submit.

 

·         If employee’s name is changed, mail a copy of appropriate legal document to Main Office.

 

·         Once this request is approved by main office, employee’s new information will be updated to your Aloha system automatically.

 

 

 

 

Permanent Employee Transfer window:                                                  á

 

 

 

·         Use this window to transfer an employee permanently to another location.

 

·         Department and/or Pay Rate can also be changed.

 

·         If employee is stopping at some other location for training or some other reason for few days before joining the new location, enter the information under Temporary Store section.

 

·         If employee is not available to sign write “n/a” in your signature pad.

 

·         All fields with pink color are mandatory.

 

·         Once the Main Office approves this request, the employee will be added to the new location’s Aloha system automatically. Employee will also be removed from your store’s Aloha system permanently after the end of the current pay period.

 

 

 

 

 

 

 

 

Employee Separation window:                                                                  á

 

 

 

·         Use this window to separate an employee.

 

·         All fields in pink color are mandatory.

 

·         Check box Mail Final Wages to Following Address, if you want the payroll department to mail a final check to the employee. An email will be sent to the payroll department with a mail final wages request as soon as you submit this request. Keep this box unchecked if the employee received a Cash Paid Out.

 

·         Check Will Get Signature Later box, if you want to get all signatures later. You will have to go to Update Requests queue to sign the request.

 

·         If employee is not available to sign, write “n/a” on signature pad. 1st Witness Manager’s Signature and 2nd Witness Manager’s Signature are required.

 

·         Once the Main Office approves this request, this employee will be removed from your Aloha system automatically after the end of the current pay period.

 

 

 

Edit Punch:                                                                                                          á

 

 

·         Use this window to modify hourly employees’ punches.

 

·         Select Date, Department and click on Get Punch Data. All employees worked in that department on the selected date will appear in the grid.

 

·         Select the employee you want to modify punch, with mouse and then click Edit Shift, a separate window will pop up with the punch details.

 

·         After selecting employee in the grid, you can also click Add New Shift to add another shift for that employee.

 

·         Click View Punch Details & Calculation to view report of the selected employee’s punch data and over time calculations for the whole period.

 

·         Edit Shift Window

 

 

  • Click on Add Break button if you want to add additional breaks. Add breaks first before you make any changes on this window, as adding break will undo all the changes you made.

 

  • Make changes if any to Department worked, Dec Tips or Sales. If you change any of these, you will have to select Edit Reason.

 

  • Make changes if any to Punch Details, Break. You will have to select Edit Reason for every line you change.

 

  • All hours are in hh:mm format. If Time OUT is less than Time IN, it will be considered next day.

 

  • Column Time Elapsed displays time difference between Time IN and Time OUT

 

  • After done with the changes, click Calculate Total Paid Time to get total paid time. Column Time Elapsed will be updated with the changes too.

 

  • After done with all changes, click Update to save your changes. All changes will be reflected in the grid on the Edit Punch window.

 

  • Click Reset to start over again. Window will go to the position when you opened it first time.

 

 

 

·         Add New Shift Window

 

 

 

  • Click on Add Break button if you want to add breaks. Add breaks first before you make any changes on this window, as adding break will undo all the changes you made.

 

  • Enter Date Worked, select Department and Time IN, Time OUT for Punch Details and breaks if any.

 

  • All hours are in hh:mm format. If Time OUT is less than Time IN, it will be considered next day.

 

  • Click Calculate Total Paid Time to get total paid time. Column Time Elapsed will be updated with the time differences too.

 

  • Click Update to add shift or Reset to start over.

Submit Payroll:                                                                                          á

 

 

·         When done with punch edits, use this window to submit payroll.

 

·         It displays hour summary for all departments, with over time calculations.

 

·         Click Submit Payroll to submit payroll for displayed pay period. You can submit payroll only once. Once submitted, next pay period will be displayed.

 

·         For payroll details click View Payroll Summary button, a report window will pop up where you can see hours and over time calculations for each employee.

 

 

 

 

 

 

Update Requests:                                                                                       á

 

 

·         If you have made any Quick Hire or Quick Separation (Separation request without signatures) request, they will go in Update Requests queue.

 

·         You will have to update every request here. GM or AGM can not view the request in their Process Requests queue until you update the request here.

 

·         GM or AGM can deny any Quick Hire request, if the employee doesn’t show up without updating.

 

 

 

 

 

 

 

 

 

 

Reports:                                                                                                     á

 

 

·         Use this to view different requests’ reports made from your store and their status.

 

·         Select Date Created or Date Effective and enter Start Date, End Date and click View Report, report will appear in separate window.

 

·         Use other list boxes (Approval Status, Post Status) to further filter reports.

 

·         Date Created is the date request created from your store and Date Effective is the date the request will be effective.

 

·         It is recommended that you view the report before making any request to avoid duplicate requests.

Process Requests :                                                                                     á

(Approve / Deny Employee Status Change Requests window)      

 

 

 

·         All requests (except Name/Add Change) made by OA or Managers will go in the queue for GM/AGM Approval.

 

·         Click on the Process link in first column to process request. Request will pop up in a new window with Approve and Deny buttons at the bottom.

 

·         Requests in pink color are quick requests, which need to be updated from Update Requests queue.

 

·         Once approved or denied, the request will be removed from the queue.

 

·         GM/AGM has to approve the request before the Main Office can process it further. GM/AGM can make any changes to the request before approval.

 

·         If GM/AGM denies the request, it will be removed permanently from the queue.

 

·         Since OA, Managers and AGM can view only hourly employees of the store, GM will have to make status change requests for all salaried employees of the store as explained in the first half of this document.
